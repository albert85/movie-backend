# MOVIE LIST BACKEND

## Installation
- Clone the repo
- Create `.env` file and add the folowing
  - DATABASE_URL: mongodb database link
  - TOKEN_PASSWORD: password for encoding and decoding jwt token

- Start this project

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the project on port 8080
